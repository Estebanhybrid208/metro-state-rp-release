## Overview

Metro State RP is a FiveM-driven roleplay community website focused on organized, story-based interaction. It is built for players who want a common community hub for law enforcement, civilian, fire, and EMS roleplay, with tools and presentation choices that support structured departments and ongoing participation.

This project is intended for communities that need a polished public-facing website to represent their server identity and broader roleplay environment. With custom vehicles, distinct departments, and active staff support, Metro State RP provides members with a clear place to connect, browse roles, and stay involved in the community.
